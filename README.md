# JobQuery
*Currently in development*

A simple job search application.

This tool's purpose is to help drive the development and refinement of types in **Project Galahad** for a v0.1 release - similar to SpecDemo and the other applications. It is easier to evolve and refactor types via these small applications rather than trying to rework the whole library at once.

The evolving **Project Galahad** types are located in the Base folder.
